Basit Matematik Oyunu - Unity

Bu proje, Unity ile geliştirilmiş temel bir matematik oyunudur. Oyun, oyuncuya rastgele iki sayı ve dört işlemden (`+`, `-`, `*`) birini sunar. Oyuncu doğru cevabı klavye üzerinden girip `Enter` tuşuna bastığında, oyunun doğru mu yoksa yanlış mı olduğunu belirtir.

 🎮 Oyun Özellikleri

- Rastgele iki sayı üretimi
- Rastgele işlem seçimi (`+`, `-`, `*`)
- Kullanıcıdan cevap alma
- Doğru veya yanlış geri bildirimi

 🛠️ Kullanılan Teknolojiler

- Unity (2020 ve üzeri önerilir)
- C# (Unity Scripting)

 🚀 Nasıl Çalışır?

1. Oyun başladığında ekranda bir matematik işlemi gösterilir (örneğin `7 * 3`).
2. Oyuncu cevabı giriş kutusuna yazar ve `Enter` tuşuna basar.
3. Oyun, girilen cevabın doğru olup olmadığını kontrol eder ve ekranda "Doğru!" veya "Yanlış!" mesajını gösterir.
4. Ardından yeni bir işlem oluşturularak oyun devam eder.

Simple Math Game - Unity

This project is a basic math game developed with Unity. The game presents the player with two random numbers and one of the three basic operations (`+`, `-`, `*`). The player enters their answer using the keyboard and presses the `Enter` key. The game then indicates whether the answer is correct or incorrect.

 🎮 Game Features

- Random number generation
- Random operator selection (`+`, `-`, `*`)
- User input for answers
- Feedback for correct or incorrect answers

 🛠️ Technologies Used

- Unity (version 2020 or newer recommended)
- C# (Unity scripting)

 🚀 How It Works

1. When the game starts, a math operation is displayed (e.g., `7 * 3`).
2. The player types the answer into the input field and presses the `Enter` key.
3. The game checks whether the answer is correct and displays a "Correct!" or "Wrong!" message.
4. A new problem is then generated, and the game continues.
